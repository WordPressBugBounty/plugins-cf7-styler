== Description ==

CF7 WOW Styler helps you style your Contact Form 7 forms visually, without writing CSS. You design directly in the browser, see every change in the live preview, and apply the style with one click.

Use CF7 WOW Styler to adjust colors, spacing, buttons, backgrounds (including animated backgrounds), lists, labels, and error messages. The free version is ideal if you want to style one important form, for example your main contact or request form.
